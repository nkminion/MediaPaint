MediaPaint

A web-based drawing tool that lets you paint on a digital canvas using hand gestures. Powered by MediaPipe.


Pre-Requisites:

Allow webcam access: The webpage requires webcam access to determine the user's gestures.


Gestures Used:

Index-Finger = Hover/Select : Hover over a button for a while to select it.

Index and Middle Finger = Draw/Erase


Made With:

HTML5 and CSS3: Structure and Styling

JavaScript: Main Logic

MediaPipe: Computer Vision
